# Description of Bock 1997 data

A 3-item tabulated data set extracted from Table 3 in Chapter Two.

## References

Bock, R. D. (1997). The Nominal Categories Model. In van der Linden, W.
J. & Hambleton, R. K. *Handbook of modern item response theory*. New
York: Springer.

## Author

Phil Chalmers <rphilip.chalmers@gmail.com>
